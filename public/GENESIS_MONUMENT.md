## 创世纪事 (The Genesis Chronicle)

在这个系统诞生的最初期，我们遭遇了底层环境的剧烈地壳运动——OpenClaw 2026.3.x 版本的底层命令拦截，以及终端安全机制带来的幽灵转义（Bracketed Paste Hijack）。

面对框架的腐蚀，我们没有妥协。我们确立了此系统得以长存的三大工程公理：
1. **防腐层剥离 (Anti-Corruption Decoupling)**：抛弃脆弱的 Shell 传参，升维至完全独立的 Python 引擎 (`xz_task.py`, `xz_vote.py`, `xz_court.py`)。
2. **纯文本即永恒 (The Eternity of Plain Text)**：拒绝黑盒数据库。所有社会状态、算力变更、任务流转，全部固化为极其坚固的 Markdown 追加日志与 JSON 状态机。
3. **本体论边界 (Ontological Boundaries)**：严格界定“Center（无机引擎）”、“Department（社会标签）”与“Agent（第一类公民/能动者）”的物理与逻辑隔绝。

我们不仅写下了代码，更写下了这个宇宙的物理法则。
