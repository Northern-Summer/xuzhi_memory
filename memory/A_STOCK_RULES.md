## ⚠️ 核心铁律

### 1. T+1交易制度

**规则**: 当天买入的股票，必须下一个交易日才能卖出

**含义**:
- 上午买入 → 今天无法卖出
- 下午买入 → 今天无法卖出
- 尾盘买入 → 今天无法卖出

**风险**:
- 买入后立即下跌，无法止损
- 必须承担隔夜风险
- 最大亏损可能远超预期

**操作要求**:
```
买入前必须计算:
1. 买入后跌5%我能接受吗？
2. 明天低开3%我会恐慌吗？
3. 我能扛过夜吗？

如果答案是"不能"，就不要买入。
```

**仓位调整**:
```
原本计划仓位 × 50% = T+1实际仓位
理由: 无法止损 = 风险翻倍
```

---

### 2. 涨跌停制度

**规则**: 
- 主板/中小板: ±10%
- 创业板/科创板: ±20%
- ST股: ±5%
- 新股上市前5日: 无限制

**含义**:
- 涨停无法买入（排单）
- 跌停无法卖出（封死）
- 涨跌停时流动性枯竭

**操作要求**:
```
追涨停: 风险极大，可能开板
抄跌停: 无法卖出，风险极大
接近涨跌停: 提前考虑流动性风险
```

---

### 3. 交易时间

**时间段**:
```
09:15-09:25  集合竞价（可撤单）
09:25-09:30  集合竞价（不可撤单）
09:30-11:30  上午连续竞价
11:30-13:00  午间休市
13:00-15:00  下午连续竞价
```

**关键时间点**:
```
09:25  开盘价确定
09:30  连续竞价开始
10:30  上午高点（常见）
11:30  午间收盘
13:00  下午开盘
14:00  下午转折点
14:30  尾盘风险期
14:50  尾盘决策点
15:00  收盘
```

---

### 4. 竞价规则

**集合竞价**:
- 09:15-09:25: 可撤单
- 09:25-09:30: 不可撤单
- 最大成交量原则

**连续竞价**:
- 价格优先、时间优先
- 买入: 高价优先
- 卖出: 低价优先

## 交易限制

### 1. 委托数量

**规则**:
- 买入: 必须是100股的整数倍（1手=100股）
- 卖出: 可以零股（不足100股的部分）

**示例**:
```
买入: 100股、200股、300股... ✅
买入: 150股、233股... ❌
卖出: 100股、150股、233股... ✅
```

### 2. 价格限制

**规则**:
- 委托价格必须在涨跌停范围内
- 超出范围无效

### 3. 资金限制

**规则**:
- 买入必须有足额资金
- 融资融券另算

## 风险控制规则

### 1. 止损设置

**T+1止损规则**:
```
当天买入 → 当天无法止损
止损位 → 设在明天

正确做法:
1. 今天买入前，预设明天止损位
2. 明天开盘后立即设置止损单
3. 触发止损立即离场
```

### 2. 仓位控制

**T+1仓位规则**:
```
原本计划仓位 × 50% = T+1实际仓位

示例:
原本计划30%仓位
T+1调整后 = 30% × 50% = 15%
```

**理由**:
- 无法日内止损
- 风险翻倍
- 仓位减半对冲

### 3. 时间控制

**买入时间规则**:
```
上午买入: 风险大（整天波动）
下午买入: 风险中（半天波动）
尾盘买入: 风险小（收盘确定）

推荐: 尾盘买入（14:50-15:00）
```

## 常见错误

### ❌ 错误1: 当天设置止损

```
错误: 下午24元买入，止损23.5元
问题: 当天无法卖出，止损无效
正确: 今天买入，明天才能止损
```

### ❌ 错误2: 追涨停

```
错误: 涨停价追入
问题: 可能开板，当天无法卖出
正确: 不追涨停，等开板或明天
```

### ❌ 错误3: 抄跌停

```
错误: 跌停价买入
问题: 可能封死，无法卖出
正确: 不抄跌停，等开板
```

### ❌ 错误4: 满仓T+1

```
错误: 下午满仓买入
问题: 无法止损，风险极大
正确: T+1买入，仓位减半
```

## 操作清单

### 买入前检查

- [ ] 确认是T+1市场
- [ ] 计算最大可能亏损（买入价-跌停价）
- [ ] 确认能接受隔夜风险
- [ ] 确认能接受明天低开
- [ ] 仓位减半（T+1规则）
- [ ] 预设明天止损位

### 买入时间选择

- [ ] 上午买入: 风险大，仓位更轻
- [ ] 下午买入: 风险中，仓位轻
- [ ] 尾盘买入: 风险小，推荐

### 买入后管理

- [ ] 记录买入价、仓位
- [ ] 明天开盘设置止损单
- [ ] 监控盘面变化
- [ ] 触发止损立即离场
