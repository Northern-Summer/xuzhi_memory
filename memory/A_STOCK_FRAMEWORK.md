## 一、分析前检查清单

### 1. 规则检查

```
□ T+1规则: 当天买入无法卖出
□ 涨跌停: 涨停无法买入，跌停无法卖出
□ 委托数量: 买入100股整数倍
□ 交易时间: 9:30-11:30, 13:00-15:00
□ 资金充足: 确认可用资金
```

### 2. 风险计算

```
□ 最大亏损 = (买入价 - 跌停价) / 买入价
□ 隔夜风险 = 今晚消息面 + 明天开盘
□ 流动性风险 = 接近涨跌停时的买卖难度
□ 仓位风险 = T+1仓位减半
```

### 3. 时间规划

```
□ 买入时间: 尾盘最优（14:50-15:00）
□ 观察时间: 关键时间点（10:30, 14:00, 14:30）
□ 止损时间: 明天开盘后才能设置
```

## 二、买入决策流程

### Step 1: 确认市场环境

```
大盘: 涨/跌/震荡
板块: 强/弱/分化
个股: 强/弱/跟随
```

### Step 2: 计算风险收益比

```
上行空间: 目标价 - 当前价
下行风险: 当前价 - 止损价
风险收益比 = 下行风险 / 上行空间

要求: 风险收益比 < 1:2
```

### Step 3: T+1风险调整

```
原始仓位: X%
T+1调整: X% × 50% = X/2%

示例:
原始计划30% → T+1调整15%
```

### Step 4: 设置止损位

```
止损位: 明天开盘后设置
止损幅度: -3% 到 -5%
止损方式: 集合竞价或开盘后立即挂单
```

### Step 5: 选择买入时间

```
优先级:
1. 尾盘（14:50-15:00）: 最优，风险最小
2. 下午（13:30-14:30）: 中等，仓位更轻
3. 上午（9:30-11:30）: 风险大，仓位最轻
```

## 三、卖出决策流程

### Step 1: 确认持仓状态

```
买入日期: 必须T+1才能卖出
持仓成本: 买入价 + 手续费
当前盈亏: (当前价 - 成本) / 成本
```

### Step 2: 触发条件

```
止盈触发: 达到目标价
止损触发: 跌破止损位
时间触发: 到达预设时间
信号触发: 技术指标/板块变化
```

### Step 3: 卖出执行

```
集合竞价: 9:15-9:25挂单
连续竞价: 交易时间挂单
跌停封死: 无法卖出，等待开板
```

## 四、仓位管理规则

### 基础仓位

```
单只股票: 不超过总资金30%
同一板块: 不超过总资金50%
总仓位: 根据市场环境调整
```

### T+1调整

```
当天买入: 原始仓位 × 50%
理由: 无法止损，风险翻倍
```

### 时间调整

```
上午买入: 仓位 × 50% × 50% = 25%
下午买入: 仓位 × 50%
尾盘买入: 仓位 × 50% × 100% = 50%
```

### 示例

```
原始计划: 30%仓位
上午买入: 30% × 25% = 7.5%
下午买入: 30% × 50% = 15%
尾盘买入: 30% × 50% = 15%
```

## 五、止损规则

### 止损位设置

```
技术止损: 支撑位下方2-3%
资金止损: 最大亏损3-5%
时间止损: 买入后2-3天不涨则离场
```

### T+1止损

```
当天买入: 无法止损
明天开盘: 立即设置止损单
触发止损: 立即离场，不加仓
```

### 止损执行

```
集合竞价: 9:15挂止损单
开盘观察: 如果大幅低开，等待反弹
严格执行: 触发必须离场
```

## 六、常见错误警示

### ❌ 错误1: 忘记T+1

```
错误: 当天设置止损
正确: 止损位设在明天
```

### ❌ 错误2: 仓位过重

```
错误: T+1满仓
正确: T+1仓位减半
```

### ❌ 错误3: 追涨停

```
错误: 涨停价追入
正确: 不追涨停，等开板或明天
```

### ❌ 错误4: 忽视隔夜风险

```
错误: 下午买入不看晚间消息
正确: 考虑晚间消息面风险
```

### ❌ 错误5: 没有止损计划

```
错误: 买入后不设止损
正确: 买入前预设止损位
```
