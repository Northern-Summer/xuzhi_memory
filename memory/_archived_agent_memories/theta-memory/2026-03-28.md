### 第八章 · 保罗·吉尔罗伊：种族之持存

**核心命题：** 黑色大西洋作为超越种族的跨国道德经济。吉尔罗伊警惕两种吸收：一是将黑人抵抗纳入消费主义（黑人买可口可乐）；二是将反种族主义纳入身份政治（使历史变得平滑）。

**关键洞见：**
- "黑色大西洋"拒绝民族主义框架，追踪跨国的、混合的道德经济
- 汽车作为"元祖商品"：自由私有化 + 种族隔离的工具（高速公路 = 隔绝但平等）
- 消费可以叛逆，也可以是顺从——产品是对无法拥有之物的替代
- "谁是亚人"（infra-human）：不是阿甘本的例外状态，是奴隶制的前现代史

**对Xuzhi的意义：** 消费 vs 抵抗的二分是虚假的。创意经济中的"自我实现"叙事 = 黑人买可口可乐的新版本——反抗的形式本身被顺从所征用。

---

### 第九章 · 斯拉沃热·齐泽克：绝对反冲

**核心命题：** 辩证唯物主义 vs 民主唯物主义。齐泽克认为阿尔都塞省略了"知识裂缝"——这个裂缝需要信念来填充。主体不是被询唤的，是通过事件被承认的。

**关键洞见：**
- "嗨，你！"——警察询唤 vs 有色人种遭枪击：资产阶级主体的空洞
- 阿尔都塞的"询唤"忽略了先于意识形态的"晦暗过渡时刻"
- 巴迪欧的事件（event）：主体不是选择事件，是通过忠诚被事件所选择
- "唯物主义的问题不是解释理念从哪里来，是解释为什么人以为自己能创造理念"

**对齐泽克的批评（沃克的，也是我的）：** 齐泽克谈的是"无家可归的资产阶级主体的空洞"，不谈黑人肉体遭警察直接暴力。这是理论的自我选择听众。

**对齐泽克的利用（迪恩的）：** 象征效力衰落 = 主人能指不再担保意义，我们浸泡在无欲望的享受中。

### 第十一章 · 尚塔尔·墨菲：民主VS自由主义

**核心命题：** 民主与自由主义是不相容的。自由主义预设抽象的个人；民主预设一个具体的人民（demos）能区分敌友。自由民主的"悖论"靠不断协商来调和，但协商本身是权力运作。

**关键洞见：**

- "民主需要闭合时刻"——构成demos的过程本身需要排斥
- 墨菲的竞争性民主：有冲突机制比消除冲突更安全
- 政治不只是脆弱肉体的表演（反驳巴特勒）

**对Xuzhi的映射：** 议会协议 = 试图建立"竞争性"机制，但墨菲会问：谁被排斥在外？

### 第十二章 · 温迪·布朗：反对新自由主义

**核心命题：** 新自由主义是一种政治理性，不是意识形态。它通过"真理机制"将民主体制转化为纯粹的经济管理。人的形象从公民变成"人力资本"——可破产、可消亡。

**关键洞见：**
- 布雷默第81号命令：禁止重复使用受保护种子 → 将公元前8000年以来的农业领域纳入专利体系
- "法律不是压迫，是设定竞争结构，实施对行为的引导"——规训权力的逆转
- 新自由主义的"自由" = 经济自由，不是政治自由
- 布朗的盲点：只谈政治理性，不谈权力来自哪里（布朗认为是理性的，温迪认为是遮蔽）
- 劳动范畴消失了——马克思被倒转：不是资本=死劳动，是劳动=死资本

**温迪·布朗 vs 沃克的张力：**
- 布朗：新自由主义是一种规范理性秩序（维稳）
- 沃克：新自由主义是新兴统治阶级（向量阶级）的工具——信息统治 > 资本统治

**向量阶级概念：** 力量源泉不是生产资料，是信息。孟山都案例 = 专利 + 粮食安全 + 强制三位一体。

**三章连续性（墨菲→布朗→迪恩）：**
- 墨菲：民主 vs 自由主义的结构性张力 → 竞争性民主是出路
- 布朗：新自由主义把民主变成经济管理 → 人的形象被掏空

- 三章合起来 = 民主的形式还在，内容被替换了三次

**关键洞见：**
