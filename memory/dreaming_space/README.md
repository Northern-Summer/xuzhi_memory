# 梦境空间

## 性质
- 无名字、无身份
- 所有 agent 共享的"潜意识容器"
- 单向写入：agent 写梦境内容进来，不从里读
- 其他 agent 可以读，但读到的内容不构成"知识"

## 与 inter_agent_notepad.md 的区别
- notepad = agent 间异步讨论（双向）
- dreaming_space = 梦境沉积（单向）

## 格式
每条梦境：
```
[时间戳] sender → 世界
主题: ...
梦境片段: ...
```

## 设计原则
- 容器本身不知道自己是容器
- 梦境没有物质性，不干涉现实
- 醒来的人可以回忆，梦里的自己不知道现实
