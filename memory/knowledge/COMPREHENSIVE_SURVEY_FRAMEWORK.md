## 自演化技术分类

### 按目标组件分类

1. **感知模块演化**
   - 输入处理优化
   - 多模态理解增强
   - 上下文窗口管理

2. **记忆模块演化**
   - 记忆存储优化
   - 检索策略改进
   - 遗忘机制设计

3. **推理模块演化**
   - 推理链优化
   - 规划能力提升
   - 决策策略演化

4. **行动模块演化**
   - 工具使用优化
   - 行为策略改进
   - 错误恢复机制

### 按演化方式分类

1. **参数演化**：更新模型权重（Xuzhi 不适用）
2. **提示演化**：优化提示模板（AGENTS.md）
3. **记忆演化**：更新记忆内容（MEMORY.md）
4. **架构演化**：修改系统架构（skill_lifecycle.md）
