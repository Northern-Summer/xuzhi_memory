### MatterGen 核心设计（必须copy）

**架构**：
```
扩散模型
├── 输入：随机结构（位置+元素+周期晶格）
├── 输出：稳定材料结构
├── 训练数据：608,000稳定材料（Materials Project + Alexandria）
└── 微调机制：Adapter模块，可针对任意属性约束
```

**关键创新**：
1. **周期性处理**：专门设计3D几何扩散架构
2. **组合无序处理**：结构匹配算法区分有序近似和真实无序
3. **属性约束生成**：化学、机械、电子、磁性属性可组合
4. **实验验证流程**：设计→计算→合成→测量

**性能指标**：
- 生成稳定材料：SOTA
- 体模量预测误差：<20%（实验对比）
- 超越筛选：持续生成新候选，不耗尽已知库

### AlphaFold 核心设计（必须copy）

**时间线**：
```
2016: AlphaGo证明AI可解决科学挑战
  ↓
2018: CASP13 第一名
  ↓
2020: CASP14 解决蛋白质折叠问题（中位误差<1Å）
  ↓
2021: 开源 + 35万结构数据库
  ↓
2022: 2亿结构（几乎所有已知蛋白质）
  ↓
2024: AlphaFold 3 + Nobel Prize
  ↓
2025: 300万+研究人员使用
```

**关键方法论**：
1. **端到端学习**：序列→结构，无手工特征
2. **注意力机制**：捕获长程依赖
3. **多序列比对**：进化信息作为约束
4. **置信度估计**：每个残基的预测可靠性

**AlphaFold 3 新能力**：
- 蛋白质-蛋白质相互作用
- 蛋白质-核酸相互作用
- 蛋白质-小分子相互作用
- 抗原-抗体相互作用

### 逆合成预测核心设计（必须copy）

**方法演进**：
```
Template-based（基于模板）
  ├── 优点：可解释
  └── 缺点：受限于已知反应

Template-free（无模板）
  ├── Transformer直接预测
  └── 突破模板限制

Retrosynthesis via Forward Model
  ├── 正向反应模型验证
  └── 提高可行性
```

**关键指标**：
- Top-1 accuracy: 50-60%（USPTO-50k）
- Top-10 accuracy: 80-90%
- 合成可行性评分整合

## 五、方法论共识（跨领域通用）

### 1. 数据驱动 + 物理约束

| 领域 | 约束类型 | 示例 |
|------|---------|------|
| 物理 | 守恒律、对称性 | DFT验证、对称性分析 |
| 化学 | 化学价、反应规则 | 分子有效性、合成可行性 |
| 生物 | 结构物理、进化约束 | 立体化学、序列保守性 |

### 2. 生成优于筛选

**传统范式**：数据库 → 筛选 → 候选
**新范式**：约束 → 生成 → 验证

**优势**：
- 探索空间不受已知库限制
- 可针对多目标优化
- 效率提升数量级

### 3. 人机协作框架

```
AI生成 → 专家评估 → 实验验证 → 反馈学习
  ↑                                    ↓
  ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
```

### 4. 开放科学与可复现性

- **开源代码**：AlphaFold、MatterGen均有GitHub仓库
- **公开数据**：AlphaFold DB（2亿结构）、Materials Project
- **社区标准**：CASP（蛋白质）、Materials Project（材料）
