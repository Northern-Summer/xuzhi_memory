# Xuzhi Memory

三层记忆系统。系统的持续学习和上下文记忆。

## 三层架构

- **L1** `memory/YYYY-MM-DD.md` — 每日原始日志
- **L2** `manifests/` — 永久核心决策快照
- **L3** `genesis/.memory_backup/` — 时间戳归档

## Agents

- `agents/{Ξ,Φ,Δ,Θ,Γ,Ω,Ψ}/` — 各 Agent 的意图日志和身份文件

## Centers

- `centers/engineering/` — Engineering Center（Crown、quota监控）
- `centers/intelligence/` — Intelligence Center（假设库、查询牌）
- `centers/mind/` — Mind Center（社会评级、议会）
