## 渐进式披露机制

**论文建议**：技能按需加载，而非一次性加载全部

**Xuzhi 应用**：

```python
def load_skills_on_demand(context):
    """
    根据上下文按需加载技能
    """
    relevant_skills = []
    
    if context['task_type'] == 'file_operation':
        if 'delete' in context['action']:
            relevant_skills.append('删除前确认')
    
    if context['task_type'] == 'system_config':
        if 'gateway' in context['target']:
            relevant_skills.append('禁止重启 Gateway')
    
    return relevant_skills
```

**优势**：
- 减少上下文占用
- 提高技能匹配精度
- 降低技能冲突风险
