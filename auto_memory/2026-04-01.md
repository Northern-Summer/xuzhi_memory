## 自动提取逻辑

```python
def extract_learning(user_message, response):
    """
    从对话中提取学习内容
    """
    learnings = []
    
    # 检测纠正
    if "记住" in user_message or "下次" in user_message:
        learnings.append({
            "type": "correction",
            "content": extract_key_point(user_message),
            "timestamp": now()
        })
    
    # 检测偏好
    if "我喜欢" in user_message or "我希望" in user_message:
        learnings.append({
            "type": "preference",
            "content": extract_preference(user_message),
            "timestamp": now()
        })
    
    return learnings
```
