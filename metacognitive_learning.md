## 核心框架

### 三大组成部分

```
┌─────────────────────────────────────────────────────────┐
│  1. 元认知知识 (Metacognitive Knowledge)                │
│  ┌───────────────────────────────────────────────────┐  │
│  │ • 能力自我评估：我知道我能做什么，不能做什么       │  │
│  │ • 任务理解：我知道任务需要什么能力                 │  │
│  │ • 学习策略：我知道如何学习新能力                   │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  2. 元认知规划 (Metacognitive Planning)                 │
│  ┌───────────────────────────────────────────────────┐  │
│  │ • 决定学什么：识别能力缺口                         │  │
│  │ • 决定怎么学：选择学习策略                         │  │
│  │ • 分配学习资源：时间、工具、数据                   │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  3. 元认知评估 (Metacognitive Evaluation)               │
│  ┌───────────────────────────────────────────────────┐  │
│  │ • 反思学习经验：什么有效，什么无效                 │  │
│  │ • 改进学习策略：更新策略库                         │  │
│  │ • 预测未来学习：估计学习难度和时间                 │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

## 内在 vs 外在元认知

### 外在元认知（当前主流）

```
Human 设计的固定循环：
  检测失败 → 提取技能 → 写入知识库
```

**局限**：
- 循环是固定的，不会自己演化
- 学什么、怎么学是 Human 决定的
- 无法适应新任务域

### 内在元认知（目标）

```
Agent 自主的动态循环：
  自我评估能力 → 识别缺口 → 选择策略 → 学习 → 反思 → 调整策略
       ↑                                                        ↓
       ←←←←←←←←←←←← 学习过程本身也在学习 ←←←←←←←←←←←←←←←←←←←←←←
```

**Xuzhi 当前进度**：
- ✅ 已有自我评估机制（进化测量）
- ✅ 已有缺口识别机制（Session End 失败检测）
- ✅ 已有反思机制（ECHO 校准）
- 🟡 学习策略选择：目前是手动的，需要自动化
- 🟡 学习过程元学习：尚未实现
