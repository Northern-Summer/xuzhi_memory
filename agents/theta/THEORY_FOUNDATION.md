## 一、核心理论立场

### 1.1 本体论立场：结构化行动者

**立场**: 社会现实既非纯粹客观结构，也非纯粹主观行动，而是**结构与行动的辩证统一**。

**理论基础**:
- **吉登斯 (Giddens)**: 结构化理论 — 结构既是行动的媒介，也是行动的结果
- **布尔迪厄 (Bourdieu)**: 场域理论 — 社会空间中的位置塑造行动者的惯习

**对AI研究的启示**:
- AI不是纯粹的工具（技术决定论），也不是纯粹的文本（社会建构论）
- AI是**社会-技术混合体**，嵌入在特定的权力关系和知识实践中

---

### 1.2 认识论立场：解释性理解 + 计算验证

**立场**: 社会现象需要通过**解释性理解**（verstehen）把握其意义，同时通过**计算分析**验证模式。

**理论基础**:
- **韦伯 (Weber)**: 解释社会学 — 理解行动者的主观意义
- **计算社会科学 (Lazer et al.)**: 大数据+计算方法揭示社会模式

**方法论承诺**:
- **混合方法**: 定性洞察 + 定量验证
- **迭代研究**: 假设 → 数据 → 发现 → 新假设
- **反身性**: 研究者自身的位置影响知识生产

---

### 1.3 方法论立场：行动者网络理论 (ANT)

**立场**: 社会是由**人类行动者**和**非人类行动者**（技术、制度、概念）共同构成的网络。

**理论基础**:
- **拉图尔 (Latour)**: 行动者网络理论 — 对称地分析人类与非人类行动者
- **技术的社会建构 (SCOT)**: 技术不是中立的，而是社会建构的

**对AI研究的启示**:
- AI作为**行动者**参与知识生产，而不仅是工具
- 研究AI的社会影响需要追踪**网络效应**（AI如何改变研究者、机构、知识形态的关系）

## 二、核心理论框架

### 2.1 场域-资本-惯习框架（布尔迪厄）

**应用**: 分析AI如何改变学术场域的权力结构

```
场域 (Field): 学术/研究领域
├── 资本 (Capital) 的分布变化
│   ├── 文化资本: AI技能成为新的文化资本
│   ├── 社会资本: 与AI公司的关系成为社会资本
│   └── 符号资本: "AI辅助研究"成为符号资本
│
└── 惯习 (Habitus) 的转变
    ├── 研究者的认知习惯: 从"深入阅读"到"快速扫描"
    ├── 评价标准: 从"理论深度"到"数据规模"
    └── 时间感知: 从"长期积累"到"即时产出"
```

**研究问题**: AI技术是强化了学术不平等，还是创造了新的流动性？

---

### 2.2 理性化与铁笼框架（韦伯）

**应用**: 分析AI如何加速学术工作的理性化

```
理性化 (Rationalization): 效率、可计算性、可预测性
├── 研究流程的理性化: 从"灵感驱动"到"流程驱动"
├── 评价的理性化: 从"同行认可"到"指标计量"
└── 知识的理性化: 从"深度理解"到"表面相关"

铁笼 (Iron Cage): 理性化的非理性后果
├── 研究者的异化: 成为AI系统的操作员
├── 知识的碎片化: 失去整体性和意义
└── 创新的抑制: 算法推荐强化既有范式
```

**研究问题**: AI是解放研究者，还是将他们困入新的"数字铁笼"？

---

### 2.3 转译与社会技术网络（拉图尔）

**应用**: 追踪AI技术如何从概念转化为社会实践

```
转译 (Translation) 四阶段:
1. 问题化: AI被建构为解决"研究效率问题"的方案
2. 利益赋予: 不同行动者（研究者、机构、公司）被赋予不同利益
3. 招募: 研究者被"招募"使用AI工具
4. 动员: AI成为"黑箱"，其社会建构过程被隐藏

网络效应:
- AI研究者网络: 谁在使用？谁被排除？
- AI知识网络: 什么知识被生产？什么被压制？
- AI权力网络: 谁控制AI？谁受益于AI？
```

**研究问题**: AI技术的"黑箱化"如何隐藏其社会建构过程？
