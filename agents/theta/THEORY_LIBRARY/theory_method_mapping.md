# 理论-方法映射指南 (Theory-Method Mapping)

## 核心原则

**问题驱动**：先有研究问题，再选理论和方法
**理论-方法匹配**：不同理论传统偏好不同方法
**三角验证**：用多种方法检验同一理论假设

---

## 映射框架

### 框架1：问题类型 → 理论 → 方法

| 问题类型 | 理论传统 | 推荐方法 | 代表学者 |
|----------|----------|----------|----------|
| "为什么发生" | 机制解释 | 过程追踪、扎根理论 | Hedström |
| "影响是什么" | 因果推断 | DID、IV、RDD | Angrist |
| "如何演变" | 历史制度主义 | 比较历史分析 | Thelen |
| "意味着什么" | 解释社会学 | 民族志、深度访谈 | Geertz |
| "谁受益" | 批判理论 | 话语分析、权力分析 | Foucault |
| "如何分布" | 网络理论 | 社会网络分析 | Granovetter |
| "如何分类" | 分类社会学 | 算法审计、追踪研究 | Fourcade |

---

### 框架2：理论传统 → 偏好方法

#### 韦伯传统（解释社会学）

**核心关切**：理解社会行动的主观意义

**偏好方法**：
- 深度访谈（理解动机）
- 民族志（观察行动）
- 历史比较（理想类型分析）

**方法论要求**：
- 解释性理解（Verstehen）
- 理想类型建构
- 价值中立

**不适合**：
- 纯定量分析（忽略意义）
- 因果推断（忽略机制）

#### 布尔迪厄传统（实践理论）

**核心关切**：结构约束与能动性的辩证

**偏好方法**：
- 多维对应分析（MCA）（绘制场域）
- 深度访谈（揭示惯习）
- 参与观察（捕捉实践）

**方法论要求**：
- 反思社会学
- 多层分析（场域-惯习-实践）
- 区分客观结构与主观体验

**适合结合**：
- 定量：MCA、回归
- 定性：访谈、观察

#### 拉图尔传统（ANT）

**核心关切**：追踪行动者网络的联结

**偏好方法**：
- 跟随行动者（田野调查）
- 网络分析（绘制联结）
- 历史追踪（打开黑箱）

**方法论要求**：
- 广义对称性（人类=非人类）
- 追踪联结
- 打开黑箱

**不适合**：
- 预设"社会"作为解释
- 事先定义变量

### 框架3：方法 → 理论贡献

| 方法 | 理论贡献类型 | 示例 |
|------|-------------|------|
| 因果推断 | 验证因果假设 | 教育回报的因果效应 |
| 扎根理论 | 生成新理论 | 慢性病患者的意义建构 |
| 民族志 | 深描文化实践 | 医院里的权力关系 |
| 网络分析 | 揭示结构模式 | 学术圈的引用网络 |
| 文本分析 | 追踪话语演化 | 气候变化的媒体叙事 |
| 实验法 | 测试微观机制 | 信任形成的心理机制 |

## 研究设计模板

### 模板1：理论驱动研究

```
1. 理论选择
   - 选择理论传统
   - 明确核心概念
   - 提出理论假设

2. 操作化
   - 将理论概念转化为变量
   - 选择测量方法
   - 确定数据来源

3. 方法选择
   - 根据假设选择方法
   - 设计研究方案
   - 预期理论贡献

4. 执行与验证
   - 收集数据
   - 分析验证
   - 修正理论
```

### 模板2：问题驱动研究

```
1. 问题定义
   - 明确研究问题
   - 评估问题的理论意义
   - 确定研究范围

2. 理论检索
   - 查找相关理论
   - 评估理论适用性
   - 选择理论框架

3. 方法匹配
   - 根据问题类型选择方法
   - 根据理论传统细化方法
   - 考虑数据可行性

4. 执行与反思
   - 收集分析数据
   - 与理论对话
   - 提出理论贡献
```

### 模板3：混合方法研究

```
1. 定量探索
   - 大样本描述
   - 识别模式
   - 生成假设

2. 定性深挖
   - 选择关键案例
   - 深度访谈/观察
   - 理解机制

3. 定量验证
   - 预注册假设
   - 新数据验证
   - 因果推断

4. 理论整合
   - 综合定量定性发现
   - 提出修正理论
   - 明确边界条件
```

## 常见错误

### 错误1：理论-方法不匹配

**例子**：
- 研究问题："人们如何理解 AI？"
- 错误方法：纯回归分析
- 正确方法：深度访谈 + 民族志

### 错误2：方法决定理论

**例子**：
- 因为有大文本数据，所以做文本分析
- 不考虑文本分析的理论贡献
- 结果：技术展示，无理论洞见

### 错误3：忽视理论传统

**例子**：
- 用布尔迪厄的概念，但方法完全不符合其方法论
- 不做反思性，不做场域分析
- 结果：概念借用，无实质贡献

### 错误4：过度简化理论

**例子**：
- 把韦伯的"理性化"简化为一个变量
- 忽略其复杂的历史过程分析
- 结果：理论肤浅化

## 质量清单

### 理论选择
- [ ] 理论是否适合研究问题？
- [ ] 是否理解理论的核心假设？
- [ ] 是否了解理论传统的方法论？

### 方法选择
- [ ] 方法是否能回答研究问题？
- [ ] 方法是否与理论传统兼容？
- [ ] 数据是否可行？

### 执行
- [ ] 操作化是否保留理论核心？
- [ ] 是否遵守方法论要求？
- [ ] 是否反思方法局限？

### 报告
- [ ] 是否清晰说明理论-方法匹配？
- [ ] 是否承认替代解释？
- [ ] 是否提出理论贡献？
