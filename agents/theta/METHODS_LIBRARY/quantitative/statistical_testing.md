# 统计检验 (Statistical Testing)

## 定义
统计检验是一种推断方法，通过样本数据推断总体参数，评估观察到的效应是否超出随机变异范围。

## 适用场景
- 假设检验
- 组间比较
- 效应估计
- 控制虚假发现

## 核心概念

### 1. 假设检验框架
- **零假设 (H0)**：无效应/无差异
- **备择假设 (H1)**：有效应/有差异
- **显著性水平 (α)**：拒真错误概率（通常 0.05）
- **统计功效 (1-β)**：正确拒绝错误 H0 的概率

### 2. p 值争议
- **p 值定义**：在 H0 为真时，观察到当前或更极端结果的概率
- **常见误解**：
  - ❌ p 值 = H0 为真的概率
  - ❌ p < 0.05 = 发现真实效应
  - ❌ p > 0.05 = 无效应
- **正确理解**：p 值是衡量数据与 H0 不一致程度的指标

### 3. 新统计范式（前沿）
- **效应量**：标准化效应大小（Cohen's d, r, η²）
- **置信区间**：参数估计的不确定性范围
- **贝叶斯因子**：比较 H0 和 H1 的证据强度
- **错误发现率 (FDR)**：多次比较时的校正

## 主要检验方法

### 1. 参数检验（假设正态分布）
| 检验 | 适用场景 |
|------|---------|
| t 检验 | 两组均值比较 |
| ANOVA | 多组均值比较 |
| 线性回归 | 连续因变量预测 |
| 皮尔逊相关 | 连续变量相关性 |

### 2. 非参数检验（无分布假设）
| 检验 | 适用场景 |
|------|---------|
| Mann-Whitney U | 两组秩次比较 |
| Kruskal-Wallis | 多组秩次比较 |
| Spearman 相关 | 秩相关 |
| 置换检验 | 任意统计量的显著性 |

### 3. 多重比较校正
| 方法 | 特点 |
|------|------|
| Bonferroni | 最保守，控制 family-wise error |
| Holm | 较宽松，逐步校正 |
| Benjamini-Hochberg | 控制 FDR，更宽松 |
| 置换校正 | 保留相关性结构 |

## 操作步骤

### 1. 检验前诊断
- 检查正态性（Shapiro-Wilk, QQ plot）
- 检查方差齐性（Levene's test）
- 检查样本量是否足够

### 2. 选择检验方法
- 根据数据类型和分布假设选择
- 非参数检验更稳健但功效较低

### 3. 执行检验
- 计算检验统计量
- 计算 p 值和效应量
- 计算置信区间

### 4. 报告结果（APA 格式）
```
t(58) = 2.15, p = .035, d = 0.56, 95% CI [0.12, 1.00]
```

### 5. 稳健性检验
- 不同方法交叉验证
- 敏感性分析
- 排除异常值重分析

## 反事实推理（关键检验）

### 检验1：p 值操纵
- **如果**反复尝试不同模型直到 p < 0.05
- **那么**虚假发现率急剧上升
- **对策**：预注册、报告所有分析、调整 α

### 检验2：样本量不足
- **如果**样本量太小
- **那么**功效低，真实效应可能不显著
- **对策**：先验功效分析确定样本量

### 检验3：多重比较
- **如果**进行多次检验但不校正
- **那么**至少有一次假阳性的概率接近 1
- **对策**：FDR 校正、预注册假设

### 检验4：假设违反
- **如果**数据不满足正态性假设
- **那么**参数检验结果不可靠
- **对策**：非参数检验、自助法、数据变换

## 前沿方法（2024-2026）

### 1. 贝叶斯统计
- **贝叶斯因子**：证据强度量化
- **后验分布**：参数不确定性完整描述
- **工具**：Stan, PyMC, brms

### 2. 自助法 (Bootstrap)
- 重抽样估计不确定性
- 无分布假设
- 小样本友好

### 3. 置换检验
- 构建经验零分布
- 灵活、稳健
- 适用复杂设计

### 4. 预注册和开放科学
- 先验注册假设和分析计划
- 减少选择性报告
- 工具：OSF, AsPredicted

## 工具栈
- **Python**: scipy.stats, statsmodels, pingouin
- **R**: base R, car, emmeans, BayesFactor
- **贝叶斯**: Stan, PyMC, JAGS
- **元分析**: metafor, meta

## 局限
- p 值常被误解和滥用
- 显著 ≠ 重要
- 假设检验框架局限性
- 小样本功效低
- 模型假设可能违反

## AI辅助应用
- **自动检验选择**：根据数据类型推荐方法
- **假设诊断**：自动检查正态性、方差齐性
- **结果解释**：AI 辅助撰写统计报告
- **⚠️ 警告**：
  - AI 可能过度依赖 p 值
  - 统计显著性 ≠ 实际重要性
  - 研究者必须理解统计原理
