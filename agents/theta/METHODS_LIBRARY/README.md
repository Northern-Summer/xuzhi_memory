## 方法选择指南

### 当研究问题涉及"意义""过程""机制"时 → 定性方法
- 例：研究者如何理解AI工具的角色？
- 推荐：深度访谈 + 扎根理论

### 当研究问题涉及"因果关系""模式""效应"时 → 定量方法
- 例：AI使用是否提高了研究效率？
- 推荐：因果推断 + 统计检验

### 当研究问题涉及"大规模文本""模式识别"时 → 计算方法
- 例：AI如何改变学术写作的风格？
- 推荐：NLP + 文本分析

### 混合方法（推荐）
- 先用定性探索，再用定量验证
- 或：定性深入理解 + 计算扩展规模

## 使用协议

1. **方法服务于问题**：先明确研究问题，再选择方法

4. **持续更新**：每次使用新方法后更新本库
