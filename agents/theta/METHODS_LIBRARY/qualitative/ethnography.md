# 民族志 (Ethnography)

## 定义
民族志是一种定性研究方法，研究者长期沉浸在被研究群体的自然环境中，通过参与观察和深度访谈，理解群体的文化、行为和意义系统。

## 适用场景
- 研究群体文化和亚文化
- 理解日常实践和意义建构
- 探索组织和社会群体内部运作
- 研究问题涉及"文化""实践""互动"

## 操作步骤

### 1. 进入田野
- **选址**：确定研究群体/组织
- **进入策略**：守门人协商、滚雪球抽样
- **伦理审批**：知情同意、匿名保护

### 2. 参与观察
- **参与程度**：
  - 完全观察者（局外人）
  - 观察者即参与者（边缘参与）
  - 参与者即观察者（积极成员）
  - 完全参与者（局内人）
- **观察内容**：
  - 物理环境
  - 社会互动
  - 日常例行
  - 非常规事件
  - 时间和空间使用

### 3. 田野笔记
- **描述性笔记**：客观记录观察
- **反思性笔记**：研究者思考、感受、困惑
- **方法论笔记**：研究过程记录
- **频率**：每天记录，及时整理

### 4. 深度访谈
- 半结构化访谈
- 非正式交谈
- 焦点小组（谨慎使用）

### 5. 数据分析
- 持续编码（扎根理论可结合）
- 主题分析
- 叙事分析
- 文本分析（田野笔记）

### 6. 离开田野
- 逐步退出
- 成员检验
- 数据存档

## 质量标准
- **可信性**：长期沉浸、三角验证、成员检验
- **可转移性**：厚描述
- **可依赖性**：审计轨迹
- **可确认性**：反思性备忘录

## 反事实推理（关键检验）

### 检验1：进入策略的偏见
- **如果**只通过守门人进入
- **那么**可能只接触到守门人希望你看到的一面
- **对策**：多渠道进入、滚雪球到边缘成员

### 检验2：参与程度的影响
- **如果**参与程度过高（完全参与者）
- **那么**可能"本土化"，失去分析距离
- **对策**：定期离开田野、写反思笔记、与同行讨论

### 检验3：观察者效应
- **如果**研究者存在影响被观察者行为
- **那么**数据可能反映"表演"而非真实实践
- **对策**：长期沉浸、建立信任、多方法验证

### 检验4：选择性观察
- **如果**只关注符合预期的事件
- **那么**可能错过反常现象（反常往往最有价值）
- **对策**：记录一切、定期检视假设

## 局限
- 时间消耗极大（数月至数年）
- 研究者主观性难以完全控制
- 研究情境不可复制
- 外部效度有限
- 伦理风险（隐私、知情同意）

## AI辅助应用（前沿）

### 计算民族志 (Computational Ethnography)
- **数字田野**：在线社区、社交媒体观察
- **网络分析**：结合社会网络分析
- **文本挖掘**：分析大规模文本数据
- **自动编码**：AI 辅助田野笔记编码

### 虚拟民族志 (Virtual Ethnography)
- **线上参与观察**：沉浸数字空间
- **多地点民族志**：追踪跨空间实践
- **工具**：
  - 网络爬虫（合规）
  - 社交媒体 API
  - 数字足迹分析

### ⚠️ 警告
- AI 无法替代长期沉浸的文化理解
- 数字田野需遵守平台规则和伦理
- 自动编码需研究者审核和调整
- 核心洞见来自研究者的"文化翻译"
