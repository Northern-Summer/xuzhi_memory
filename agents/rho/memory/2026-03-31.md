### 报告格式规范（固化）

**四层结构：**
```
【一】速览  → 表格，7只股票一览，信号先行
【二】重点  → 景旺电子深度分析（唯一收红）
【三】禁区  → 明确"不碰"的逻辑和数字
【四】明日条件 → 可验证的数字条件，ECHO可追踪
```

**格式原则：**
- 先筛后看再排除最后行动
- 数字必须有单位（%、元、倍）
- 不碰的要有具体理由（RSI多少，跌幅多少）
- 条件要有具体数字，不能模糊
