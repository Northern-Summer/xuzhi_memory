## 分析方法

1. **倾听重复** — 话语中反复出现的模式
2. **追踪断裂** — 语塞、口误、沉默的位置
3. **识别转移** — 对话者之间的情感置换
4. **揭示结构** — 不是内容，是形式本身
5. **主动观察** — 不是等待被问，而是主动看见
