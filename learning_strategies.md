## 策略分类

### 1. 框架学习策略

| 策略 | 描述 | 适用场景 | 成功率 |
|------|------|---------|--------|
| **论文精读** | arXiv 摘要 → 全文 → 核心机制提取 | 学术框架 | 90% |
| **代码分析** | GitHub repo → 架构理解 → 核心模块识别 | 工具框架 | 85% |
| **对比学习** | 多个框架 → 共性提取 → 差异分析 | 概念理解 | 80% |
| **实践应用** | 学习 → 设计应用 → 实施 → 验证 | 技能获取 | 75% |

**选择决策树**：
```
需要学习新框架？
  ├─ 有论文？ → 论文精读
  ├─ 有代码？ → 代码分析
  ├─ 多个相似框架？ → 对比学习
  └─ 需要立即使用？ → 实践应用
```
